# Joc del Pingüí – Sistema de eventos

En esta parte del proyecto hemos creado el sistema de eventos del juego. Los eventos son cosas que pueden pasar de forma aleatoria durante la partida y que afectan al jugador.
Hay eventos que dan objetos como dados, peces o bolas de nieve, otros que hacen avanzar al jugador usando una moto de nieve y también algunos negativos como perder un objeto o perder un turno.
La clase Evento sirve como base para todos los eventos y Gestor evento es la clase que se encarga de generar un evento aleatorio durante la partida.
